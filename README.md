## Features
- Support Laravel App
- PHP 8.1
- Nginx
- Ext. MSSQL
- Ext. MongoDB
- Mysql 8
- phpMyAdmin
# How To Prepare Environment
- Install Docker and Docker Compose
- Clone This Repo First 
- Clone Your Laravel App and Create with Folder Name "web" 
- Run Below Command on Current Directory
```bash
    docker-compose up -d
```
